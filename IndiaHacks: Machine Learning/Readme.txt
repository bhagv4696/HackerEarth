
To preprocess trainning data:
   (1) put "Preprocess_train.py" file in same folder as trainning csv files
   (2) run "Preprocess_train.py" which creates as csv file names "train_pre.csv"

To preprocess testing data:
   (1) put "Preprocess_test.py" file in same folder as testing csv files
   (2) run "Preprocess_test.py" which creates as csv file names "test_pre.csv"

After preprocessing:
   (1) put "train_pre.csv","test_pre.csv" and "ml.py" in same folder
   (2) run "ml.py" which creates submission file
